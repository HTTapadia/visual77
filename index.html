About Visual Studio Code!!


Visual Studio Code is a brilliant coding software in which you can develop games, apps, websites and everything and anything you can imagine. It is an IDE software in which you can use CSS, Javascript and Html. Brackets is another alternative however Visual Studio Code is more advanced and easier to access. With many extensions to add and libraries, you can optimize the use of Visual Studio Code at your own preference.
